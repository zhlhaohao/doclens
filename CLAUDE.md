# Cortex CLAUDE.md

Cortex — 结构感知文档检索工具

## 测试命令

```bash
source .venv/Scripts/activate                              # 激活虚拟环境
python -m cortex search <query>                           # 搜索文档
python -m cortex ai <message>                             # AI 对话
python -m cortex index [--force]                          # 重建索引（--force 全量重建）
python -m cortex status                                   # 查看状态
```
