# CRISPR基因治疗逆转肺癌化疗耐药性

## 重大突破：编辑免疫细胞攻克耐药难题

2025年，美国ChristianaCare健康系统旗下的Gene Editing Institute宣布了一项里程碑式的研究成果：利用CRISPR-Cas9基因编辑技术成功逆转了非小细胞肺癌（NSCLC）的化疗耐药性。该研究在《Nature Medicine》上发表，为晚期肺癌患者带来了新的希望。

## 研究背景

非小细胞肺癌占所有肺癌病例的85%，其中约60%的患者在化疗过程中发展出耐药性。耐药性的主要机制包括药物外排泵过表达、DNA修复能力增强和细胞凋亡通路抑制。传统应对策略如更换化疗方案或增加剂量往往效果有限且副作用严重。

## 核心技术：CISH基因编辑

研究团队将目光投向了肿瘤浸润淋巴细胞（Tumor Infiltrating Lymphocytes, TIL）疗法。TIL疗法通过提取患者肿瘤组织中的免疫细胞，在体外扩增后重新输回患者体内，以攻击肿瘤。

关键创新在于使用CRISPR-Cas9敲除TIL细胞中的**CISH基因**（Cytokine-inducible SH2-containing protein）。CISH是T细胞受体（TCR）信号通路的负调控因子，其缺失可显著增强T细胞的抗肿瘤活性：

- T细胞增殖能力提升3.5倍
- 干扰素-γ（IFN-γ）分泌量增加280%
- 肿瘤杀伤效率提高4.2倍
- 对耐药肿瘤细胞的识别能力恢复至接近化疗敏感水平

## 临床试验数据

2025年启动的I/II期临床试验（NCT05838249）入组了32名晚期NSCLC患者，这些患者均对铂类化疗产生耐药性。初步结果显示：

| 指标 | 结果 |
|------|------|
| 客观缓解率（ORR） | 41.2% |
| 疾病控制率（DCR） | 73.5% |
| 中位无进展生存期（PFS） | 7.8个月 |
| 中位总生存期（OS） | 15.3个月 |
| 严重不良事件发生率 | 8.8% |

与传统化疗方案相比，CRISPR-TIL疗法在耐药患者中的客观缓解率提升了近3倍。

## 技术优势

该疗法具有多项技术优势：

1. **自体细胞来源**：使用患者自身TIL细胞，避免移植物抗宿主病（GVHD）
2. **精准编辑**：CRISPR-Cas9编辑效率达92%，脱靶率低于0.1%
3. **持久响应**：编辑后的TIL细胞在体内存活时间超过6个月
4. **联合用药潜力**：可与PD-1抑制剂联合使用，协同效应显著

## 未来展望

研究团队计划在2026年启动III期临床试验，将入组扩大至200名患者。同时，该技术平台正在扩展到其他实体瘤类型，包括黑色素瘤、肾细胞癌和结直肠癌。

基因编辑疗法的成本控制也是研究重点。目前单次治疗费用约为35万美元，研究团队希望通过优化制备流程将成本降至15万美元以下，使更多患者受益。
