# Cortex Search Page - 前端设计指导文件

> 基于 `search_page_design.md` 需求，提供 5 种视觉风格方案供选择。
> 技术栈：Gradio (Python Web UI)

---

## 一、页面功能结构（所有风格通用）

### 1.1 布局分区

```
┌─────────────────────────────────────────┐
│          顶部导航 / 标签区               │
│   [搜索]  [AI对话]   ← 标签切换          │
├─────────────────────────────────────────┤
│          搜索操作区                      │
│   [输入框........................] [搜索] │
├─────────────────────────────────────────┤
│          搜索结果列表区                  │
│   找到 X 个结果                          │
│   ┌─ 结果卡片 ──────────────────────┐   │
│   │ 文件路径                         │   │
│   │ 结果标题                         │   │
│   │ 正文片段预览...                   │   │
│   │ 评分: [████████░░] 85% | 匹配: 2/2│   │
│   └──────────────────────────────────┘   │
│   ┌─ 结果卡片 ──────────────────────┐   │
│   │ ...                              │   │
│   └──────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 1.2 组件清单

| 组件 | 说明 |
|------|------|
| 标签栏 | 「搜索」与「AI 对话」两个标签，当前选中高亮 |
| 搜索框 | 单行输入框 + 搜索按钮 |
| 结果统计 | "找到 X 个结果" 文本 |
| 结果卡片 | 文件路径 + 标题 + 正文片段 + 评分进度条 |
| 评分条 | 可视化进度条 + 百分比 + 匹配词数 |

### 1.3 交互行为

| 交互 | 行为 |
|------|------|
| 点击标签 | 切换搜索/AI对话视图 |
| 输入关键词 + 点击搜索 | 触发检索，显示结果 |
| 点击结果卡片 | 跳转到对应文档对应行 |
| 无结果 | 显示空状态提示 |
| 加载中 | 显示骨架屏或加载动画 |

## 二、5 种风格方案

---

### 风格 A：Swiss Minimal（瑞士极简风）

**定位**：专业、干净、高可读性，适合文档检索工具的严肃感

#### 视觉特征

| 属性 | 规范 |
|------|------|
| 整体感觉 | 大量留白、严格的网格对齐、功能至上 |
| 圆角 | 0-4px（几乎直角） |
| 阴影 | 无阴影，用细线边框分隔 |
| 动效 | 极少，仅 200ms 过渡 |

#### 色彩方案

| 角色 | 色值 | 预览 |
|------|------|------|
| 背景 | `#FFFFFF` | 白色 |
| 前景/文字 | `#000000` | 纯黑 |
| 主色/强调 | `#2563EB` | 蓝色 |
| 次要文字 | `#64748B` | 灰色 |
| 边框 | `#E2E8F0` | 浅灰 |
| 卡片悬浮 | `#F8FAFC` | 近白 |
| 标签激活 | `#000000` 下划线 | 纯黑 |

#### 字体

| 用途 | 字体 | 字号/字重 |
|------|------|-----------|
| 标题/标签 | Inter / System Sans | 16-18px / 600 |
| 正文 | Inter / System Sans | 14px / 400 |
| 文件路径 | JetBrains Mono / Monospace | 12px / 400 |
| 评分数字 | Tabular Nums | 13px / 500 |

#### 组件细节

- **标签栏**：纯文字标签，选中项底部 2px 黑色下划线，无背景色变化
- **搜索框**：1px `#E2E8F0` 边框，聚焦时变为 `#000000`，无圆角
- **结果卡片**：底部 1px `#E2E8F0` 分隔线，无背景色，悬浮时 `#F8FAFC`
- **评分条**：细长进度条（4px 高），填充色 `#2563EB`，背景 `#E2E8F0`

#### 适合场景
追求极致专业感和高密度信息展示，开发者/研究人员用户群

### 风格 B：Bento Grid（便当盒网格风）

**定位**：Apple 风格现代感，模块化卡片，视觉层次丰富

#### 视觉特征

| 属性 | 规范 |
|------|------|
| 整体感觉 | 柔和圆角卡片、微妙阴影、模块化组织 |
| 圆角 | 16-20px（大圆角） |
| 阴影 | 柔和多层阴影 `0 4px 12px rgba(0,0,0,0.06)` |
| 动效 | 卡片悬浮 scale(1.01) + 阴影扩展，250ms |

#### 色彩方案

| 角色 | 色值 | 预览 |
|------|------|------|
| 页面背景 | `#F5F5F7` | Apple 浅灰 |
| 卡片背景 | `#FFFFFF` | 纯白 |
| 前景/文字 | `#1D1D1F` | Apple 深灰 |
| 主色/强调 | `#0D9488` | Teal 绿 |
| 次要文字 | `#86868B` | Apple 灰 |
| 边框 | `#D2D2D7` | 柔和灰 |
| 评分填充 | `#0D9488` | Teal 渐变 |
| 标签激活背景 | `#0D9488` + 白字 | Teal 填充 |

#### 字体

| 用途 | 字体 | 字号/字重 |
|------|------|-----------|
| 标题/标签 | Plus Jakarta Sans | 16-18px / 600 |
| 正文 | Plus Jakarta Sans | 14px / 400 |
| 文件路径 | SF Mono / Monospace | 12px / 400 |
| 评分数字 | Tabular Nums | 13px / 600 |

#### 组件细节

- **标签栏**：圆角胶囊标签，选中项 Teal 实色背景 + 白字
- **搜索框**：20px 圆角，白色背景 + 柔和阴影，聚焦时阴影加深
- **结果卡片**：20px 圆角白色卡片，16px 间距，悬浮 scale(1.01)
- **评分条**：8px 高圆角进度条，填充色带从 `#14B8A6` 到 `#0D9488` 渐变

#### 适合场景
面向大众用户的现代感设计，重视觉质感和亲和力

### 风格 C：Dark Cinema（深色影院风）

**定位**：沉浸式深色主题，适合长时间阅读，高端科技感

#### 视觉特征

| 属性 | 规范 |
|------|------|
| 整体感觉 | 深邃背景、微光效果、玻璃质感、环境光晕 |
| 圆角 | 16px |
| 阴影 | 玻璃边框 `rgba(255,255,255,0.08)` + 微妙顶部高光 |
| 动效 | Spring 物理动画，250-400ms，缓动贝塞尔 |

#### 色彩方案

| 角色 | 色值 | 预览 |
|------|------|------|
| 页面背景 | `#050506` → `#020203` 渐变 | 深黑渐变 |
| 卡片背景 | `rgba(255,255,255,0.05)` | 半透明 |
| 前景/文字 | `#EDEDEF` | 亮灰白 |
| 主色/强调 | `#5E6AD2` | Indigo 靛蓝 |
| 次要文字 | `#8A8F98` | 柔和灰 |
| 边框 | `rgba(255,255,255,0.08)` | 微光边框 |
| 评分填充 | `#5E6AD2` + 光晕 | 靛蓝发光 |
| 标签激活 | `#5E6AD2` 底部指示条 | 靛蓝高亮 |

#### 字体

| 用途 | 字体 | 字号/字重 |
|------|------|-----------|
| 标题/标签 | Inter / System Sans | 16-18px / 600 |
| 正文 | Inter / System Sans | 14px / 400 |
| 文件路径 | JetBrains Mono | 12px / 400, `#8A8F98` |
| 评分数字 | Tabular Nums | 13px / 600 |

#### 组件细节

- **标签栏**：深色背景标签，选中项底部 2px `#5E6AD2` 指示条 + 微弱文字光晕
- **搜索框**：16px 圆角，`rgba(255,255,255,0.05)` 背景 + 微光边框，聚焦时边框 `#5E6AD2` 发光
- **结果卡片**：16px 圆角半透明卡片，顶部 1px 高光线，悬浮时边框增亮
- **评分条**：6px 高进度条，`#5E6AD2` 填充 + 外发光 `rgba(94,106,210,0.3)`

#### 适合场景
开发者/技术用户，长时间使用场景，护眼优先

## 四、通用设计规范（所有风格共享）

### 4.1 图标规范
- 使用 SVG 图标（Lucide 图标库），禁止使用 Emoji
- 图标尺寸：16px（行内）、20px（按钮）、24px（标签）
- 描边宽度统一 1.5px 或 2px

### 4.2 间距系统
- 基础单位：8px
- 间距阶梯：4 / 8 / 12 / 16 / 24 / 32 / 48
- 内容最大宽度：`max-width: 800px`（搜索结果页为阅读型页面，不宜过宽）

### 4.3 排版规范
- 正文最小字号：14px
- 行高：1.5-1.75
- 正文片段最大显示：3-4 行，超出截断 + 省略号
- 文件路径显示：相对路径，过长时中间省略

### 4.4 交互规范
- 可点击元素：`cursor: pointer`
- 搜索按钮：异步操作时禁用 + 显示加载状态
- 空结果：友好提示 + 建议操作
- 键盘支持：Enter 键触发搜索

### 4.5 无障碍
- 文字对比度 >= 4.5:1（WCAG AA）
- 交互元素 focus 状态可见
- 进度条提供文字替代（百分比数值）
- 尊重 `prefers-reduced-motion`
